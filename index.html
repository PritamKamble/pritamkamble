📅 Program Structure (16-Week Core)

Each week: 6–8 hrs class + 10–12 hrs labs/self-work. Friday quiz, weekend interview prep.

Week Theme Key Topics Weekly Deliverables
W0 Orientation & Tooling Linux/CLI, Git branching/PRs, HTTP/JSON, APIs, VS Code, Nx basics Repo setup, first PR, coding standards
W1 Programming Foundations JS → TypeScript, async, modules, FP vs OOP, error handling TS kata set, utility lib with tests
W2 DSA I + CS Basics Arrays/strings/hashmaps, Big-O, OS/DBMS/Networks overview 15 problems (LC patterns), CS flashcards
W3 Frontend I HTML5/CSS3, responsive layout, Tailwind, a11y Landing page from Figma, responsive grid
W4 Frontend II React hooks, state mgmt, Next.js App Router, forms, SSR/ISR, SEO Next.js blog with auth + comments
W5 Backend I Node/NestJS, REST design, validation (zod/class-validator), auth (JWT), Prisma/Drizzle User management API with auth

Complete 16-week curriculum available upon enrollment. Includes weeks 6-15 covering Backend II, Database Design, Testing, DevOps, GenAI Integration, Cloud Deployment, and Full-Stack Projects.
